# OOP_Lab1_Chasnyk
На початку було створено консольну програму. Клас програми: авто. В програмі було меню із шести пунктів.
Далі до ЛБ 2 було скореговано програму, і додано декілька private класів
В ЛБ 3, було додано клас перевантажених конструкторів, де один з конструкторів немає параметрів(при його викорастанні застосовуєтся ініціалізатори), один конструктор реалізований через виклик іншого власного конструктора
Також для деяких класів було також додано перевантажені вірсії
